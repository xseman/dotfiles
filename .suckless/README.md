# Suckless

<https://suckless.org/>

This is my setup running on Debian 10

## Dependencies

```sh
sudo apt install make gcc libx11-dev libxft-dev libxinerama-dev xorg
```

## Install

Simple, in directory of tool just run:

```sh
sudo make install
```
